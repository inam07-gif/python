<h1>Housing Price Prediction using Machine Learning</h1>

<h2>Project Overview</h2>
<p>
This project focuses on predicting housing prices using multiple Machine Learning models.
The dataset used is the California Housing dataset. The objective is to compare different
regression algorithms and evaluate their performance.
</p>

<h2>Models Implemented</h2>
<ul>
<li>Linear Regression (with Polynomial Features)</li>
<li>K-Nearest Neighbors Regressor (KNN)</li>
<li>Decision Tree Regressor</li>
<li>Random Forest Regressor</li>
<li>Support Vector Regressor (SVR)</li>
</ul>

<h2>Evaluation Metrics</h2>
<p>The models were evaluated using the following metrics:</p>
<ul>
<li>MAE (Mean Absolute Error)</li>
<li>MSE (Mean Squared Error)</li>
<li>RMSE (Root Mean Squared Error)</li>
<li>R² Score</li>
</ul>

<h2>Results</h2>
<p>
Among all the models tested, Random Forest Regressor achieved the best performance
with the highest R² score and lower prediction errors. This indicates that ensemble
models perform well for tabular datasets with complex relationships.
</p>

<h2>Conclusion</h2>
<p>
The project demonstrates how different regression models behave on the same dataset.
By comparing multiple algorithms, the study shows that ensemble learning methods such
as Random Forest provide better predictive performance for housing price prediction.
</p>
